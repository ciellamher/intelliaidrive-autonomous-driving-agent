**intelliaidrive-autonomous-driving-agent**
A full-stack autonomous driving simulation platform combining computer vision and reinforcement learning.

### 01 — INSTALL

Bash
> git clone https://github.com/ciellamher/intelliaidrive-autonomous-driving-agent.git
> cd intelliaidrive-autonomous-driving-agent

---

### 02 — USE

Initialize the simulation environment:

> python run_simulation.py

---

### 03 — WHAT'S INSIDE

- `run_simulation.py` — The core environment controller.
- `agent.py` — Reinforcement learning models and reward policies.
